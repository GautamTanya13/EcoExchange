# EcoExchange

**Overview**  
EcoExchange is an e-commerce web application designed to facilitate the buying and selling of reusable products. This project was developed as part of my coursework for the Internet Application module in the Master of Applied Computing program. Our team received recognition for the best user interface and presentation.

**Technologies Used**  
- HTML
- CSS
- JavaScript
- Python
- Django (Framework)

**Key Features**  
1. **User Authentication**: A user-friendly interface for signup and login, leveraging Django's built-in authentication framework.
2. **Marketplace**: A comprehensive marketplace that allows users to view listings, with sorting, filtering, and search capabilities powered by various algorithms.
3. **EcoCoins**: A dedicated currency system enabling users to purchase EcoCoins through a payment gateway.
4. **User Profiles**: Individual pages for product details, seller information, order history, and coin history, created using distinct views and templates.
5. **Auction Feature**: An auction option for users, facilitated through Reddit channels.
6. **Seller Functionality**: CRUD operations for sellers to manage their listings effectively using Django forms.
7. **Admin Dashboard**: A dedicated admin interface for managing the web application, utilizing Django's built-in features.
8. **Support Features**: Sections for About Us, feedback, support, and user inquiries.

**Getting Started**  
To run the project locally, follow these steps:
1. Clone the repository using the command: `git clone https://github.com/GautamTanya13/EcoExchange.git`.
2. Install the necessary libraries listed in `requirements.txt`.
3. Start the server using the command: `python3 manage.py runserver`.
